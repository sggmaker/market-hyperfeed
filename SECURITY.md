# Security Policy

The `v0.1.0-alpha.1` security scope is local evaluation of public exchange WebSocket capture, local WAL files, and ClickHouse replay.

Report suspected vulnerabilities through GitHub private vulnerability reporting on this repository. If you are viewing a fork where that feature is unavailable, contact the repository owner through their GitHub profile and avoid posting exploit details publicly.

Do not include API keys, exchange credentials, wallet keys, private infrastructure paths, or sensitive logs in issues, discussions, screenshots, or pull requests.
